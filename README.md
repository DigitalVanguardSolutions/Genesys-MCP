# Genesys MCP

Model Context Protocol server and client for the Genesys Cloud platform — exposing Genesys capabilities (queues, conversations, users, routing, analytics, …) as MCP tools and resources so AI agents can interact with Genesys through a standard protocol.

> **Status:** project initialization. Stack, scope, and source layout are being scaffolded.
